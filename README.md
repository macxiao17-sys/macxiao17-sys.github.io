# Xiao 的个人网站

这是我的个人主页，托管在 GitHub Pages 上。

🌐 **访问地址：** https://macxiao17-sys.github.io

## 🚀 如何使用

### 添加个人照片
1. 把你本人的照片放到 `images/` 文件夹
2. 重命名为 `photo.jpg`（或修改 `index.html` 中的文件名）
3. 支持格式：JPG、PNG、WebP

### 修改内容
编辑 `index.html` 文件，修改以下部分：
- **个人信息**：名字、简介、职位
- **经历**：时间线里的内容
- **技能**：技能卡片
- **项目**：项目展示区
- **联系**：邮箱和 GitHub 链接

### 预览
直接用浏览器打开 `index.html` 即可本地预览。

## 🛠 技术栈
- HTML5
- CSS3（暗色主题、渐变、动画）
- JavaScript（滚动动画、响应式菜单）
